# Manufacturing Defect Detection
### [Industrial-AI]

A Computer Vision project focused on identifying defects in manufactured parts on a production line.

## 📁 Repository Structure
- `src/`: Deep Learning model architectures and training scripts.
- `notebooks/`: Image processing, model evaluation, and inference tests.
- `data/`: Placeholder for image datasets.
- `outputs/`: Performance metrics, confusion matrices, and saved model weights.

## 🚀 Getting Started
1. Clone the repository.
2. Install dependencies: `pip install -r requirements.txt`.
3. Check the `notebooks/` folder for model training and testing.

## 🛠️ Tools
- Python
- TensorFlow / PyTorch
- OpenCV
- Matplotlib
